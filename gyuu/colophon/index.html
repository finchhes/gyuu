WHAT IS THIS?

this is a colophon, a page dedicated to some relevant information about the way this site was made and an overview of the assets, tools, programs, and other things used.

TECHNICAL

hosted on neocities and github using deploy-to-neocities coded by hand in zed tested with zen on my 14" macbook pro m3 no cookies, though there is optional localStorage (like the autoplay toggle) javascript is used heavily, some pages require it to function
